### 2. Linux命令入门
16. 什么是命令，命令行
    - 命令：即Linux操作指令，是系统内置的程序，可以以字符的形式去使用
    - 命令行：即Linux终端，可以提供字符化的操作页面供命令执行

17. Linux命令通用格式
```
common [-options] [parameter]
commom 命令体
-options 可选选项 控制命令的行为细节
parameter 可选参数 控制命令的指向目标
```
18. ls命令的作用
    - 在命令行中，以平铺的方式，展示工作目录(默认HOME目录)下的内容(文件或是文件夹)

19. HOME目录是
    - 每一个用户在Linux系统的专属目录，默认在/home/用户名

20. 当前工作目录
    - Linux目录在执行目录的时候，需要一个工作目录，打开命令行程序(终端)默认设置工作目录在用户的HOME目录    

21. ls命令的参数的作用
    可以指定要查看的文件夹(目录)的内容，如果不给定参数，就查看当前工作目录的内容

22. ls的命令的选项

    ```
    ls [-a -l -h] [Linux路径]
    -a选项 ：可以展示出隐藏内容
    	 	- 以.开头的文件或是文件夹默认被隐藏，需要-a 才能显示出来
    -l选项：以列表的形式展示内容，并展示更多细节
    -h选项：需要和-l搭配使用，以更加人性化的方式显示文件的大小单位
    ```

23. 命令的选项组合使用
    - 命令的选项是可以组合使用的，比如ls -lah, 等同于ls -a -l -h

### 3. 目录切换相关的命令(cd/pwd)

### 4. 相对路径，绝对路径和特殊路径符
26. 相对路径和绝对路径
    - 相对路径：以根目录做起点，描述路径的方式以/开头
    - 相对路径：以当前目录做起点，描述路径的方式，路径不需要要/开头

27. 特殊连接符那些
    - . 表示当前目录，比如：cd.或cd./Desktop
    - ..表示上一级目录，比如：cd..或cd../..
    - ~表示用户的HOME目录，比如:cd~或cd~/Desktop

### 8. 运程管理
81. 什么是端口
    - 端口是计算机和外部交互的出入口，可分为物理端口和虚拟端口
      - 物理端口：USB,HDMI,DP,VAG,RG45等
      - 虚拟端口：操作系统和外部交互的出入口
    - IP只能确定计算机，通过端口才能锁定要交互的程序
    
82. 端口的划分
    - 公认端口：1~1023，用于系统内置hi哦常用知名软件绑定使用
    - 注册端口：1024~49151，用于松散的绑定使用(使用自定义)
    - 动态端口：49152~65535，用于临时使用(多用于出口)

83. 查看端口占用
    - nmap IP地址，查看指定IP的对外暴露端口
    - netsstat -anp |grep 端口号，查看本机指定端口号占用情况

    -f选项 以完全格式的形式展示信息(展示全部信息)
    固定 ps -ef 列出全部进程的全部信息
    
    从左到有分别是
    UID   进程所属的用户ID
    PID   进程的进程号ID
    PPID  进程的符ID(启动此进程的其他进程)
    C     此进程的CPU占有率(百分比)
    STIME 进程的启动时间
    TTY   启动此进程的终端序号，如显示? 表示非终端启动
    TIME  进程的占用时间
    CMD  进程对应的名称或启动路径或启动命令
    
    ps -ef | grep 关键字 过滤指定关键字进程信息
    ```

### 10. 环境变量
92. 什么是环境变量？
    - 环境变量是一组信息记录，类型是KeyValue型（名称=值），用于操作系统运行的时候记录关键信息
93. 通过env命令可以查看当前系统配置的环境变量信息
94. 通过$符号，可以取出环境变量的值
95. 什么是PATH，作用是？
环境变量PATH会记录一组目录，目录之间用:隔开。这里记录的是命令的搜索路径，当执行命令会从记录中记录的目录中挨个搜索要执行的命令并执行。
可以通过修改这个项目的值，加入自定义的命令搜索路径
如export PATH=$PATH:自定义路径
96. 如何修改环境变量？
临时生效：export 名称=值
永久生效：
针对用户：~/.bashrc文件中配置
针对全部用户：/etc/profile文件中配置
配置完成，可以通过source命令立刻生效
