### 3.时间戳
- 能获取当前的时间戳
- 使用场景：计算倒计时效果，前面方法无法计算，需要借助时间戳完成
- 什么是时间戳
  - 是指1970年01月01日00分00秒起至现在的毫秒数，是一种特殊的计量时间的方式
- 算法
  - 将来的时间戳 - 现在的时间戳 = 剩余的时间毫秒数
  - 剩余时间毫秒数转换为剩余时间的年月日时分秒就是倒计时时间
  - 比如：将来时间戳 2000ms - 现在的时间戳1000ms 1000ms
  - 1000ms转换就是0小时0分1秒

- 三种获取时间戳
  - 使用getTime()方法
    ```javascript
      const date = new Date()
      console.log(date,getTime())
    ```
  - 简写 +new Date()
    ```javascript
      console.log(+new Date())
    ```
  - 使用Date.now()
    - 无需实例化
    - 只能得到当前的时间戳，前面两种可以得到指定的时间戳
      ```javascript
        console.log(Date.now())
      ```
- 总结
  - 实例化日期对象
    - new Date()
  - 日期对象方法里有月份和星期有怎么注意的
    - 月份是0~11，星期是0~6
  - 获取时间戳有那种方法，重点记那个
    - date.getTime()
    - +new Date()
    - Date.now()
    - 重点记+new Date()可以返回当前时间戳。也可以指定时间戳

   
## 2.节点操作

### 1.DOM节点
- 目标：能够说出DOM节点的类型
- ![DOM节点](../../../imgs/js/webAPI/Day04/01-节点操作-DOM节点.png)
- DOM节点
  - DOM树里的每一个内容都称为节点
- 节点类型
  - 元素节点
    - 所有标签，比如body div
    - html是根节点
  - 属性节点
    - 所有属性节点 比如href
  - 文本节点
    - 所有的文本
  - 其他

- 总结
  - 什么是DOM节点
    - DOM树里的每一个内容都称为节点
  - DOM节点的分类
    - 元素节点 比如div标签
    - 属性节点 比如href属性
    - 文本节点 比如标签里面的文字
  - 重点
    - 元素节点
    - 可以更好的理清标签与属性的关系
### 2.查找节点
- 父节点
  - 父节点查找
  - parentNode属性
  - 返回最近一级的父节点
    ```javascript
    子元素.parentNode
    ```
- 子节点
  - childNodes
    - 获得所有子节点，包含文字节点(空格，换行)，注释节点等
  - children(重点)
    - 仅仅获得所有元素节点
    - 返回的还是有一个伪数组
    - 语法
      ```JavaScript
      父元素.children
      ``` 
- 兄弟节点
  - 根据兄弟关系查找
    - 下一个兄弟节点
      nextElementSibling
    - 上一个兄弟节点
      previousElementSibling属性

- 总结
  - 查找父节点用哪个
    - parentNode
  - 查找所有的子节点用哪个
    - children
  - 查找兄弟节点用哪个
    - nextElementSibling
    - previoudElentSibling

### 3.增加节点
- 目标：能够具备新增节点的能力
- 很多情况下，我们需要在页面中增加元素
  - 比如 点击按钮，新增一条信息
- 一般情况下，新增节点，按照一下步骤
  - 创建一个新节点
  - 把创建的新节点放入到指定元素的内部

- 创建节点
  - 创造出网页新的元素，在添加网页内，一般先创建结点，在插入
  - 创建元素结点方法
    - 语法
       ```javascript
        //创造出新的元素结点
        document。createElemenr('标签名')
       ```
- 追加结点
- 想让界面看到，还得插入某个父元素中
- 插入到父元素的最后一个子元素
  - 语法 
    ```JavaScript
      //插入到这个元素最后
      父元素.appendchild(要插入的元素)
    ```
- 插入到父元素中某个子元素的前面
  ```javascript
    //插入到某个子元素的前面
    父元素.insertBefore(要插入的元素， 在哪个元素前面)
  ```

- 在特殊情况下，我们新增节点，
  - 复制一个原有的节点
  - 把复制节点放入到指定元素的内部
- 克隆节点
  ```javascript
    //克隆一个原有的元素节点
    元素.cloneNode(Boolean)
  ```
- cloneNode克隆一个根原有标签一样的元素，括号内传入boolean
  - 若伪true，则代表克隆时会包括后代节点一起克隆
  - 若为false，则代表克隆部包括节点一起克隆
  - 默认为false

### 4.删除节点
- 目标：能够具备根据需求删除节点的能力
- 若一个节点在页面中以不需要时，可以删除它、
- 在JavaScript原生DOM中要删除元素必须通过父元素删除
- 注意
  - 如果不存在父子级关系则删除不成功
  - 删除节点和隐藏节点(display：none)有区别的:隐藏节点还是存在的，但删除，则从html中删除元素
- 语法
  ```javascript
    父元素。removeChild(要删除的元素)
  ```
