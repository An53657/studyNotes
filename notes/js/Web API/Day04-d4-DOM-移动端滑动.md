## 1.日期对象
- 目标:掌握日期对象，可以让网页显示日期
- 日期对象：用来表示时间对象
- 作用：可以得到当前系统时间

### 1.实例化
- 目标：能够实例化日期对象
- 在代码发现new实例化时，一般将这操作称为实例化
- 创建一个时间对象并获取时间
- 获取当前时间
    ```javascript
    const data = new Data()
    ```

  - 获的指定时间
    ```javascript
    const data = new Data('2020-8-8')
    ```


### 2.事件对象方法

- 日期对象
- 目标:能够使用日期对象中的方法写出常见的日期
- 使用场景：因为日期对象返回的数据不能直接使用，需要转换实际开发中常用格式


| 方法          | 作用             | 说明               |
| ------------- | ---------------- | ------------------ |
| getFullYear() | 获得年份         | 获取四位年份       |
| getMonth()    | 获得月份         | 取值为0~11         |
| getDate()     | 获取月份的每一天 | 不同月份取值也不同 |
| getDay()      | 获取星期         | 取值0~6(0为星期天) |
| getHours()    | 获取小时         | 取值0~23           |

### 3.时间戳
- 能获取当前的时间戳
- 使用场景：计算倒计时效果，前面方法无法计算，需要借助时间戳完成
- 什么是时间戳
  - 是指1970年01月01日00分00秒起至现在的毫秒数，是一种特殊的计量时间的方式
- 算法
  - 将来的时间戳 - 现在的时间戳 = 剩余的时间毫秒数
  - 剩余时间毫秒数转换为剩余时间的年月日时分秒就是倒计时时间
  - 比如：将来时间戳 2000ms - 现在的时间戳1000ms 1000ms
  - 1000ms转换就是0小时0分1秒

- 三种获取时间戳
  - 使用getTime()方法
    ```javascript
      const date = new Date()
      console.log(date,getTime())
    ```
  - 简写 +new Date()
    ```javascript
      console.log(+new Date())
    ```
  - 使用Date.now()
    - 无需实例化
    - 只能得到当前的时间戳，前面两种可以得到指定的时间戳
      ```javascript
        console.log(Date.now())
      ```
- 总结
  - 实例化日期对象
    - new Date()
  - 日期对象方法里有月份和星期有怎么注意的
    - 月份是0~11，星期是0~6
  - 获取时间戳有那种方法，重点记那个
    - date.getTime()
    - +new Date()
    - Date.now()
    - 重点记+new Date()可以返回当前时间戳。也可以指定时间戳
