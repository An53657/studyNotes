# Day14-d4-移动web-移动适配方案-vm

## vw适配方案
1. 概述
   - 相对单位
   - 相对视口的尺寸计算结果
   - vw： viewport width
     - 1vw = 1/100视口宽度

   - vh：viewport height
     - 1vh = 1/100视口高度


2. vw布局
   - 确定设计稿对应的vw尺寸 (1/100视口的高度)
     - 查看设计稿的宽度 -> 确定设备宽度(视口宽度) -> 确定vw尺寸(1/100视口宽度)

   - vw的单位尺寸 = px单位数值 / (1/100视口宽度)
