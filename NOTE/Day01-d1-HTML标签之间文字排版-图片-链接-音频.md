## 2. 标签的基本骨架
6. HTML基本骨架
   HTML基本骨架是网页模板
   
   ```html
   <!DOCTYPE html>
   <html lang="en">
   <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <title>网页标题</title>
   </head>
   <body>
       网页主体
   </body>
   </html>
   
   ```
   - HTML  整个网页
   - head  网页头部，存放给浏览器看的代码，例如CSS
   - body  网页主体，存放给用户看到的代码，例如图片，文字
   - title 网页标题

7. VS code如何快速生成HTML基本骨架
   ! (英文状态下) 配合 Enter 或 Tab

## 4. 注释
9. 什么是注释
     - 注释代码的解释说明，其目的就是让人们能够更加轻松的了解代码。注释是编写程序时，写程序的人给一个语句，程序段，函数等解释或提示，能够提高代码程序的可读性
       - 在编写HTML的代码时，我们经常在一些关键代码旁做一下注释，比如方便理解，分别查找或方便项目组里面的其他程序员了解代码，而且可以方便以后对自己的代码进行修改

   ```HTML
   <!--....--> 注释标签用来在源文档中插入注释，注释不会在浏览器中显示
   ``` 

    - 在vs code中，添加/删除注释的快捷键 Ctrl  + /

## 10. 图像标签-属性
27. 图片的属性
   ``
   属性    作用        说明
   alt     替换文本    图片玩法显示的时候显示的文字
   title    提示文本   鼠标悬停在图片上面的时候显示的文字 
   width    图片的宽带  值为数字，没有单位
   height   图片的高度  值为数字，没有单位
   ```
   - 属性写在尖括号里面，标签的后面，标签和属性名之间用括号的隔开，不区分先后顺序

## 11. 相对路径和绝对路径
28. 路径
    - 路径指的是查找文件时，从起点到终点的历经的路线
    - 路径分类 ：
      - 相对路径：从当前文件位置出发查找目录文件
      - 绝对路径：从盘符出发查找目录文件
        Windows 电脑从盘符出发
        Mac     电脑从根目录出发

29. 相对路径 从当前文件位置出发查找命令文件
    · / 表示进入某个文件夹里面  文件夹名字/
    · . 表示当前文件所在文件夹  / 

30. 在路径的写法中 以下符后表示上面含义
    .  当前文件所在的文件夹
    .. 当前文键的上一级文件夹
    /  进入某个文件夹里面

31. 绝对路径 从盘符出发查找目标文件
    - Windows电脑从盘符出发
    - Mac电脑从根目录(/)出发·
    Windows默认是\ 其他系统是/ 建议统一写·/
      ```
