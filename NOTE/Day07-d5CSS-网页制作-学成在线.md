# Day07-d5CSS-网页制作-学成在线

## 1. 项目目录
1.  网站的根目录是指存放网站的第一层文件夹，内部包括当前网站的所有素材，包括html css js 图片等
   - study
     - images文件夹：存放固定的图片素材，比如logo 样式修饰符等
     - uploads：     存放非固定的图片素材，例如：商品图宣传图需要上传的图片
     - CSS文件夹：    存放CSS文件(link引入)
       - base.css:     基础公共样式，例如清除默认样式，设置网页基本样式等
       - index.css:    首页CSS样式
     - index.css:      首页HTML文件 

2. 项目目录中，images文件夹存放什么
   - 固定使用的图片素材

3. 项目目录中，upload文件夹存放什么
   - 非固定使用的图片素材(需要上传的)

4. HTML中使用什么标签引入单独的CSS文件
   - link标签

## 2. 网页制作思路
5. 布局思路、

### 1. logo制作技巧
8. logo功能
   - 单击跳转的首页
   - 搜索引擎优化：提升网站百度搜索的排名

9. 实现方法
   - 标签结构：h1 > a > 网站名称(搜索关键词)
   - CSS样式
     ```css
     .logo a {
        display:block;
        width:195px;
        height:41px;
        background-image:url();
        <!-- 隐藏字体 -->
        font-size：0；
     }
     ```
