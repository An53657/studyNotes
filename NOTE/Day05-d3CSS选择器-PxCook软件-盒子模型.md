# Day05-d3CSS选择器-PxCook软件-盒子模型

## 1. 结构伪类选择器
1. 结构伪类选择器作用
   - 根据元素的结构关系查找元素
   ```html
   选择器             说明
   E:first-child      查找第一个E元素
   E:last-child       查找最后E元素
   E:nth-child(N)     查找第N个E元素(第一个N元素值为1)
   ```
2. E:根据元素的结构关系查找多个于元素
```html
功能                公式
偶数标签             2n
奇数标签             2n+1， 2n-1
找到5的倍数的标签     5n
找到第5个以后的标签   n+5
找到第5个以前的标签   n-5
```

3. 伪元素选择器
   - 作用创建虚拟元素(伪标签)，用来摆放装饰性的内容
   ```html 
   选择器     说明
   E::before  在E元素里面最前面添加一共伪元素
   E::after   在E元素里面最后面添加一共伪元素
   ```

   - 注意点
          - 必须设置content:""属性，用来设置伪元素的内容，如果没有内容，则引号留空即可
          - 伪元素默认为是行内显示模式
          - 权重和标签选择器相同

## 2. PxCook
4. PxCook(像素大厨)是一款切图设计工具软件，支持PSD文件文字，颜色，距离自动识别
   - 开发面版(自动智能识别)
   - 设计面板(手动测量尺寸和颜色)

## 3. 盒子模型
5. 盒子重要组成部分
   - 内容区域 - with & hight
   - 内边距   - padding(出现在内容与盒子边缘之间)
   - 边框线   - border
   - 外边框线 - margin(出现在盒子外面)

6. 边框线
   - 属性名：border (bd)
   - 属性值：边框线的粗细 线条样式 颜色 (不区分顺序)
   - 常用的属性值
   ```html
   属性值   线条样式
   solid   实线
   dashed  虚线
   dotted  点线
   ```

7. 设置当方向的边框线
   - 属性名：border-方位名词 (bd + 方位名词的首字母，例如 bdl )
   -属性值： 边框线粗细 线条样式 颜色 (不区分顺序)

8. 盒子模型-内边距
   - 作用:设置 内容 与 盒子边缘 之间的距离
   -属性名:padding /padding-方位名词

9. 盒子模型- 内边距-多值写法
   ```html
   取值个数     示例                                含义
   一个值       padding:10                         四个方向内边距均位10px
   四个值       padding:10px 20px 30px 40px        上:10px 右20px 下:30px 左:40px
   三个值       padding:10px 40px 80px             上:10px 左右:40px  下:80px
   两个值       paddingL10px 80px                  上下:10px    左右:80px
   ```
   
   - 记忆方法 从上面开始逆时针转一圈,如果当前没数值 取值跟对面一样

10. 盒子模型-尺寸计算
    - 默认情况
    盒子尺寸=内容尺寸+border尺寸+内边距尺寸

    - 结论
    给盒子加border/padding会撑大盒子

    - 解决方法
      - 手动做减法,减掉broder/padding尺寸
      - 内减模式,box-sizing:border-box

11. 盒子模型-外边距
    - 作用：来开两个盒子之间的距离
    - 属性名：margin
    - 与padding属性值，写法含义相同

12. 清除默认样式
    - 清除标签的默认样式，比如默认的内外边距

    ```html
      * {
         padding:0;
         margin:0;

         <!-- 内减模式 -->
         box-sizing:border-box;
      }

      <!-- 去掉列表的项目符号 -->
      li {
         list-style：mone；
      }
    ```

13. 盒子模型-元素益处
    - 作用：控制益处元素的内容的形式的方式
    - 属性名：overflow
    - 属性值
    ```html
    属性值        效果
    hidden        益处隐藏
    scroll        益处滚动(无论是否益处都显示滚动条位置)
    auto          益处滚动(益处才显示滚动条位置)
    ```

14. 外边距问题-合并现象
    - 场景：垂直排列的兄弟标签，上下margin会合并
    - 想象：取两个margin中较大值生效

15. 外边距问题-塌陷问题
    - 场景：父子级的标签，子级的添加上外边距会产生塌陷问题
    - 现象：导致父子级一起向下移动
    - 解决方法
      - 取消子级margin，父级设置padding
      - 父级设置overflow：hidden
      - 父级设置borer-top

16. 行内元素-内外边距的问题
    - 场景：行内元素加margin和padding，无法改变的元素的垂直问题
    - 解决方法：给行内元素添加line-heigt可以改变垂直问题

17. 盒子模型-圆角
    - 作用：设置元素的外边框为圆角
    - 属性名：border- radius
    - 属性值：数值+px/百分比
    - 提示:属性值是圆角的半径

18. 盒子模型-圆角的多值写法
    - 记忆 从左上角顺时针赋值 没有取值的角对角线取值

19. 常用应用-正圆形状
    - 给正方形盒子设置圆角属性值为 宽度的一半/50%
    - 常见应用-胶囊形状
      - 给长方形盒子设置圆角属性值 盒子高度的一半

20. 盒子模型-阴影(拓展)
    - 作用：给元素设置阴影效果
    - 属性名：box-shadow
    - 注意：
    - 属性值：X轴偏移量 Y轴偏移量 模糊半径 扩散半径 颜色 内外阴影
    - 默认是外阴影 内阴影需要添加inset
